== Description ==
این پلاگین متعلق به پلتفرم تبلیغات هایلایت هست.
این پلاگین برای درج تگ html در انتهای نوشته ها و انتهای صفحات محصول (افزونه ووکامرس) است.



== Installation ==
نصب و کار با این افزونه بسیار راحت هست
۱ -  این افزونه از مخزن وردپرس نصب و فعال کنید.
۲ - بعد نصب این افزونه صفحه افزونه از منو سمت راست به عنوان تبلیغات هایلایت  باز کنید.
۳ - دو ویرایشگر وجود دارد که اولی مربوط به نوشته ها و دومی مربوط به صفحه هر محصول است (دومین ویرایشگر در صورت فعال بودن ووکامرس نمایش داده میشود) شما میتوانید کد html تبلیغ خود را در درون این دو ویرایشگر درج کنید. ابتدا تب متن در بالا سمت چپ ویرایشگر را کلیک کنید و کد  html تبلیغ خود را در درون ویرایشگر گذاشته و دکمه ذخیره را کلیک کنید.
